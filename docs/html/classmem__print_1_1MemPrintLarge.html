Detailed Description

template<uint16_t SIZE>
class mem_print::MemPrintLarge< SIZE >

A version of MemPrint that uses uint16_t to keep its internal size parameters, so that a maximum of 65534 character can be stored.

Template Parameters
SIZEsize of internal string buffer including the NUL terminator character, the maximum is 65535, which means a maximum string length of 65534 characters.
